## Getting Started
To get started, clone or fork the repo. You will need to have npm or yarn.

### Install the dependencies
npm:
```npm
npm install
```

Yarn:
```Yarn
yarn install
```

### Development
To spin the server, run the following command.
```npm
npm run dev
```
### Build
Ready to go to production? Run:
```npm
npm run build
```
